# 霍夫曼码算法

> 原文:[https://www.javatpoint.com/huffman-coding-algorithm](https://www.javatpoint.com/huffman-coding-algorithm)

```
Huffman (C)
1\. n=|C|
2\. Q ← C
3\. for i=1 to n-1
4\. do
5\. z= allocate-Node ()
6\. x=  left[z]=Extract-Min(Q)
7\. y= right[z] =Extract-Min(Q)
8\. f [z]=f[x]+f[y]
9\. Insert (Q, z)
10\. return Extract-Min (Q)

```
