## 自动控制器

将工厂输出的实际值与期望值进行比较的设备称为自动控制器。它确定系统的偏差，并产生控制信号，将偏差减小到 0 和小值。自动控制器产生控制信号的方式称为控制动作。

## 工业控制器根据控制动作分为

1.  开关控制器
2.  比例控制器
3.  积分控制器(一)
4.  比例+积分控制器
5.  比例+微分控制器
6.  比例+积分+微分控制器
