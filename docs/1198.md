# 三地址码

> 原文：<https://www.javatpoint.com/three-address-code>

*   三地址码是中间码。它由优化编译器使用。
*   在三地址代码中，给定的表达式被分解成几个独立的指令。这些指令很容易翻译成汇编语言。
*   每个三地址代码指令最多有三个操作数。它是赋值和二元运算符的组合。

### 例子

给出一个表达式:

```

a := (-c * b) + (-c * d)

```

三地址编码如下:

```
         t1 := -c
         t2 := b*t1
         t3 := -c
         t4 := d * t3
         t5 := t2 + t4
         a := t5

```

**t** 用作目标程序中的寄存器。

三个地址码可以用两种形式表示:**四倍**和**三倍**。
