# 双向链表

> 原文:[https://www.javatpoint.com/doubly-linked-list](https://www.javatpoint.com/doubly-linked-list)

双向链表是一种复杂类型的链表，其中一个节点包含指向序列中前一个和下一个节点的指针。因此，在双向链表中，一个节点由三部分组成:节点数据、指向序列中下一个节点的指针(下一个指针)、指向上一个节点的指针(上一个指针)。图中显示了双链表中的一个示例节点。

![Doubly linked list](../Images/439c5ac40de506d30b53c0755b58301d.png)

下图显示了一个包含三个节点的双向链表，节点的数据部分的数字从 1 到 3。

![Doubly linked list](../Images/23b38cbb693588968dc56b04fb424f4d.png)

在 C 语言中，双向链表中节点的结构可以表示为:

```
struct node 
{
	struct node *prev; 
	int data;
	struct node *next; 
} 

```

第一个节点的 **prev** 部分和最后一个节点的 **next** 部分将始终包含空值，表示每个方向的结束。

在一个单链表中，我们只能在一个方向上遍历，因为每个节点都包含下一个节点的地址，并且没有它之前节点的任何记录。然而，双链表克服了单链表的这一局限性。由于列表中的每个节点都包含其前一个节点的地址，因此我们也可以通过使用存储在每个节点的前一部分中的前一个地址来找到关于前一个节点的所有细节。

## 双向链表的内存表示

下图显示了双向链表的内存表示。一般来说，双链表为每个节点消耗更多的空间，因此，导致更广泛的基本操作，如插入和删除。但是，我们可以很容易地操作列表的元素，因为列表在两个方向(向前和向后)都维护指针。

在下图中，列表的第一个元素，即 13，存储在地址 1。头指针指向起始地址 1。由于这是添加到列表中的第一个元素，因此列表**的 **prev** 包含** null。列表的下一个节点位于地址 4，因此第一个节点在其下一个指针中包含 4。

我们可以以这种方式遍历列表，直到在下一部分找到任何包含 null 或-1 的节点。

![Doubly linked list](../Images/74bcb36282519b5bf721087b85468764.png)

## 双链表上的操作

**节点创建**

```
struct node 
{
	struct node *prev;
	int data;
	struct node *next;
};
struct node *head; 

```

下表描述了关于双向链表的所有剩余操作。

| 塞内加尔 | 操作 | 描述 |
| one | [开始插入](insertion-in-doubly-linked-list-at-beginning) | 开始时将节点添加到链表中。 |
| Two | [末端插入](insertion-in-doubly-linked-list-at-the-end) | 将节点添加到链接列表的末尾。 |
| three | [在指定节点后插入](insertion-in-doubly-linked-list-after-specified-node) | 将节点添加到指定节点之后的链接列表中。 |
| four | [开头删除](deletion-in-doubly-linked-list-at-beginning) | 从列表的开头移除节点 |
| five | [末尾删除](deletion-in-doubly-linked-list-at-the-end) | 从列表末尾移除节点。 |
| six | [删除给定数据的节点](deletion-in-doubly-linked-list-after-the-specified-node) | 删除包含给定数据的节点之后的节点。 |
| seven | [搜索](searching-in-doubly-linked-list) | 将每个节点数据与要搜索的项进行比较，如果找到该项，则返回该项在列表中的位置，否则返回 null。 |
| eight | [穿越](traversing-in-doubly-linked-list) | 访问列表的每个节点至少一次，以便执行一些特定的操作，如搜索、排序、显示等。 |

## 菜单驱动程序在 C 语言中实现双链表的所有操作

```
#include<stdio.h>
#include<stdlib.h>
struct node
{
    struct node *prev;
    struct node *next;
    int data;
};
struct node *head;
void insertion_beginning();
void insertion_last();
void insertion_specified();
void deletion_beginning();
void deletion_last();
void deletion_specified();
void display();
void search();
void main ()
{
int choice =0;
	while(choice != 9)
	{
		printf("\n*********Main Menu*********\n");
		printf("\nChoose one option from the following list ...\n");
		printf("\n===============================================\n");
		printf("\n1.Insert in begining\n2.Insert at last\n3.Insert at any random location\n4.Delete from Beginning\n
		5.Delete from last\n6.Delete the node after the given data\n7.Search\n8.Show\n9.Exit\n");
		printf("\nEnter your choice?\n");
		scanf("\n%d",&choice);
		switch(choice)
		{
			case 1:
			insertion_beginning();
			break;
			case 2:
            		insertion_last();
			break;
			case 3:
			insertion_specified();
			break;
			case 4:
			deletion_beginning();
			break;
			case 5:
			deletion_last();
			break;
			case 6:
			deletion_specified();
			break;
			case 7:
			search();
			break;
			case 8:
			display();
			break;
			case 9:
			exit(0);
			break;
			default:
			printf("Please enter valid choice..");
		}
	}
}
void insertion_beginning()
{
   struct node *ptr; 
   int item;
   ptr = (struct node *)malloc(sizeof(struct node));
   if(ptr == NULL)
   {
       printf("\nOVERFLOW");
   }
   else
   {
    printf("\nEnter Item value");
    scanf("%d",&item);

   if(head==NULL)
   {
       ptr->next = NULL;
       ptr->prev=NULL;
       ptr->data=item;
       head=ptr;
   }
   else 
   {
       ptr->data=item;
       ptr->prev=NULL;
       ptr->next = head;
       head->prev=ptr;
       head=ptr;
   }
   printf("\nNode inserted\n");
}

}
void insertion_last()
{
   struct node *ptr,*temp;
   int item;
   ptr = (struct node *) malloc(sizeof(struct node));
   if(ptr == NULL)
   {
       printf("\nOVERFLOW");
   }
   else
   {
       printf("\nEnter value");
       scanf("%d",&item);
        ptr->data=item;
       if(head == NULL)
       {
           ptr->next = NULL;
           ptr->prev = NULL;
           head = ptr;
       }
       else
       {
          temp = head;
          while(temp->next!=NULL)
          {
              temp = temp->next;
          }
          temp->next = ptr;
          ptr ->prev=temp;
          ptr->next = NULL;
          }

       }
     printf("\nnode inserted\n");
    }
void insertion_specified()
{
   struct node *ptr,*temp;
   int item,loc,i;
   ptr = (struct node *)malloc(sizeof(struct node));
   if(ptr == NULL)
   {
	   printf("\n OVERFLOW");
   }
   else
   {
	   temp=head;
	   printf("Enter the location");
	   scanf("%d",&loc);
	   for(i=0;i<loc;i++)
	   {
		   temp = temp->next;
		   if(temp == NULL)
		   {
			   printf("\n There are less than %d elements", loc);
			   return;
		   }
	   }
	   printf("Enter value");
	   scanf("%d",&item);
	   ptr->data = item;
	   ptr->next = temp->next;
	   ptr -> prev = temp;
	   temp->next = ptr;
	   temp->next->prev=ptr;
	   printf("\nnode inserted\n");
   }
}
void deletion_beginning()
{
    struct node *ptr;
    if(head == NULL)
	{
		printf("\n UNDERFLOW");
	}
	else if(head->next == NULL)
	{
		head = NULL; 
		free(head);
		printf("\nnode deleted\n");
	}
	else
	{
		ptr = head;
		head = head -> next;
		head -> prev = NULL;
		free(ptr);
		printf("\nnode deleted\n");
	}

}
void deletion_last()
{
    struct node *ptr;
    if(head == NULL)
	{
		printf("\n UNDERFLOW");
	}
	else if(head->next == NULL)
	{
		head = NULL; 
		free(head);	
		printf("\nnode deleted\n");
	}
	else 
	{
		ptr = head; 
		if(ptr->next != NULL)
		{
			ptr = ptr -> next; 
		}
		ptr -> prev -> next = NULL; 
		free(ptr);
		printf("\nnode deleted\n");
	}
}
void deletion_specified()
{
	struct node *ptr, *temp;
	int val;
	printf("\n Enter the data after which the node is to be deleted : ");
	scanf("%d", &val);
	ptr = head;
	while(ptr -> data != val)
	ptr = ptr -> next;
	if(ptr -> next == NULL)
	{
		printf("\nCan't delete\n");
	}
	else if(ptr -> next -> next == NULL)
	{
		ptr ->next = NULL;
	}
	else
	{ 
		temp = ptr -> next;
		ptr -> next = temp -> next;
		temp -> next -> prev = ptr;
		free(temp);
		printf("\nnode deleted\n");
	}	
}
void display()
{
	struct node *ptr;
	printf("\n printing values...\n");
	ptr = head;
	while(ptr != NULL)
	{
		printf("%d\n",ptr->data);
		ptr=ptr->next;
	}
} 
void search()
{
	struct node *ptr;
	int item,i=0,flag;
	ptr = head; 
	if(ptr == NULL)
	{
		printf("\nEmpty List\n");
	}
	else
	{ 
		printf("\nEnter item which you want to search?\n"); 
		scanf("%d",&item);
		while (ptr!=NULL)
		{
			if(ptr->data == item)
			{
				printf("\nitem found at location %d ",i+1);
				flag=0;
				break;
			} 
			else
			{
				flag=1;
			}
			i++;
			ptr = ptr -> next;
		}
		if(flag==1)
		{
			printf("\nItem not found\n");
		}
	}	

}
```
