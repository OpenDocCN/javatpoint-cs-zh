# 词汇错误

> 原文：<https://www.javatpoint.com/lexical-error>

在词法分析阶段，可以检测到这种类型的错误。

词法错误是与任何标记模式都不匹配的字符序列。在程序执行过程中发现词法阶段错误。

## 词汇阶段错误可能是:

*   拼写错误。
*   超过标识符或数字常量的长度。
*   出现非法字符。
*   删除应该出现的字符。
*   用不正确的字符替换一个字符。
*   两个字符的换位。

### 示例:

```

Void main()
{
     int x=10, y=20;
     char * a;
     a= &x;
     x= 1xab;
}

```

在这个代码中，1ab 既不是数字，也不是标识符。所以这段代码会显示词法错误。
